## PM_BOT GERMAN



### FEATURES

The bot need to receive private messages

* Create issues
* Assign me issues
* List issues
* Filter issues
* Move issue
* Issue detail
* Notify changes on bot-channel

* Personal notification configuration
* Spring break games



### COMMANDS

* CREATE
* ASSIGN 
* ISSUE
  * blank (LIST)
  * '-t' (filter by status) example -> ISSUE -t 'In Progress'
  * ID_CODE (DETAIL)
  * ID_CODE -t COLUMN (move issue)
  * ID_CODE -u USER_NAME (assign issue to user)



### EVENTS 

* CREATED ISSUE

* ISSUE MOVEMENT

* NEW ASSIGNMENT

* CONGRATULATIONS NEW ISSUE IN PRODUCTION
